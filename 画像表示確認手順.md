# 画像表示確認手順

## 実行方法

1. **ローカルサーバーの起動**
   
   PowerShellで以下のコマンドを実行してください：
   ```powershell
   cd "C:\Users\rrryo\OneDrive\ドキュメント\Antigravity\10_ころころてんふぶ"
   npx serve .
   ```
   
   または、Python がインストールされている場合：
   ```powershell
   cd "C:\Users\rrryo\OneDrive\ドキュメント\Antigravity\10_ころころてんふぶ"
   python -m http.server 8000
   ```

2. **ブラウザでアクセス**
   
   ブラウザで以下のURLを開いてください：
   - `http://localhost:3000` (npx serve の場合)
   - `http://localhost:8000` (Python の場合)

3. **確認ポイント**
   
   - ボールに画像が表示されているか
   - F12キーを押して開発者ツールを開き、Consoleタブを確認
   - 画像の読み込み状況が表示されます：
     - ✓ Loaded: assets/ball/... と表示されれば成功
     - ❌ Failed to load: ... と表示されればエラー

4. **デバッグ情報**
   
   コンソールには以下の情報が表示されます：
   - 各画像の読み込み状況
   - ボール生成時のサイズとスケール情報
   - 全画像読み込み完了のメッセージ

## トラブルシューティング

### 画像が表示されない場合

1. **パスの確認**
   - コンソールに「Failed to load」エラーが出ていないか確認
   - assets/ball フォルダが正しい場所にあるか確認

2. **画像ファイルの確認**
   - 各フォルダに .png ファイルが存在するか確認

3. **ブラウザのキャッシュクリア**
   - Ctrl + Shift + R で強制リロード
